A plugin designed to simplify Pinterest Rich Pins integration on WordPress. 

== Description ==

Once activated, you can customize your Rich Pins in each post, adding information for three kind of items: movies, products, and recipes. 

If you need more information, check [Pinterest for Developers](http://developers.pinterest.com/rich_pins/)

[Follow me on twitter](<http://twitter.com/intent/user?screen_name=antoniobg87)

== Installation ==

1. Upload plugin files to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= I can only enter the type of pin =

You should select the type of pin and save the post. After that, when you edit the post, you can add the rest of the information for this pin

= There're some parameters that are not included (e.g. genre for movies) =
Pinterest validator doesn't validate some fields. I decided not to include these parameters. 

= Why should I enter the ingredients in English? =
Pinterest doesn't validate ingredients in other languages

= It doesn't work =
1. Make sure Pinterest validate your page.
2. Apply after validating your page.
